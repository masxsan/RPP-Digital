Capaian Pembelajaran (CP)

Peserta didik mampu memahami sistem komputer (hardware, software, brainware), membuat program sederhana, serta mengenali integrasi sistem komputer dengan teknologi AI dalam kehidupan sehari-hari.

Tujuan Pembelajaran

1Menjelaskan fungsi dan komponen sistem komputer.
2Membuat program sederhana dengan Scratch/Python.
3Menunjukkan pemahaman dasar AI.
4Menghasilkan proyek mini integrasi komputer & AI.

Langkah-Langkah Pembelajaran

Pertemuan 1: Pengenalan Sistem Komputer & Kode Program

KegiatanUraian
Pendahuluan (10 menit)
  1. Guru membuka pelajaran dengan salam dan doa.
  2. Guru menyampaikan tujuan pembelajaran hari ini.
  3. Apersepsi: guru menanyakan perangkat yang biasa digunakan siswa dan menghubungkannya dengan konsep sistem komputer.
Kegiatan Inti (40 menit)
  1. Guru menjelaskan komponen sistem komputer (hardware, software, brainware).
  2. Siswa mengamati gambar/skema sistem komputer, lalu berdiskusi kelompok untuk mengidentifikasi fungsinya.
  3. Guru memperlihatkan contoh program sederhana (Hello World) menggunakan Scratch/Python.
  4. Siswa mencoba membuat program ‚ÄúHello World‚Äù secara mandiri dengan bimbingan guru.
Penutup (15 menit)
  1. Siswa menyebutkan kembali komponen sistem komputer dan fungsinya.
  2. Guru menyimpulkan pelajaran.
  3. Pemberian tugas rumah: mencari contoh perangkat keras, lunak, dan brainware dalam kehidupan sehari-hari.

Pertemuan 2: Integrasi Sistem Komputer dengan AI

KegiatanUraian
Pendahuluan (10 menit)
  1. Guru membuka pembelajaran dengan salam.
  2. Apersepsi: guru menunjukkan contoh aplikasi AI (Google Translate, Chatbot, Teachable Machine).
  3. Guru menyampaikan tujuan pembelajaran.
Kegiatan Inti (40 menit)
  1. Guru menjelaskan pengertian Artificial Intelligence (AI).
  2. Diskusi kelompok: siswa menyebutkan manfaat AI dalam kehidupan sehari-hari.
  3. Praktik: siswa membuat proyek sederhana dengan Teachable Machine (misalnya mengenali ekspresi wajah).
  4. Presentasi hasil kerja kelompok.
Penutup (15 menit)
  1. Siswa menyampaikan kesan belajar hari ini.
  2. Guru memberikan penguatan tentang peran AI dalam sistem komputer.
  3. Menyampaikan rencana pembelajaran berikutnya dan memberi motivasi.

Lembar Kerja Peserta Didik (LKPD)

Pertemuan 1

1. Apa fungsi utama sistem komputer?

2. Sebutkan komponen sistem komputer beserta contohnya!

3. Buatlah program sederhana 'Hello World'!

4. Ubah program 'Hello World' menjadi program yang menampilkan nama Anda.

5. Buatlah program sederhana yang menampilkan penjumlahan dua angka (misalnya 5 + 3).

Pertemuan 2

1. Apa yang dimaksud dengan AI?

2. Sebutkan 3 contoh aplikasi AI!

3. Buatlah proyek sederhana dengan Teachable Machine.

4. Buatlah program sederhana yang meminta input nama, lalu menampilkan sapaan seperti: "Halo, [nama]!"

Penilaian

Asesmen Pengetahuan: Tes tertulis.

  • Sebutkan komponen utama sistem komputer!
  • Jelaskan perbedaan hardware, software, dan brainware.
  • Buatlah program sederhana untuk menampilkan teks tertentu.
  • Jelaskan pengertian AI dan sebutkan 3 contoh penerapannya.

Asesmen Keterampilan: Proyek program & integrasi AI.

  • Membuat program sederhana (Hello World, operasi matematika, input/output).
  • Proyek kelompok menggunakan Teachable Machine.

Asesmen Sikap: Keaktifan, kerjasama, kreativitas.

Rubrik Penilaian

Aspek Skor 4 (Sangat Baik) Skor 3 (Baik) Skor 2 (Cukup) Skor 1 (Kurang)
Pengetahuan Jawaban lengkap, benar, dan runtut. Jawaban cukup benar, ada sedikit kekurangan. Jawaban kurang lengkap, masih banyak kekeliruan. Jawaban tidak benar/tidak sesuai.
Keterampilan Program berjalan lancar, logika benar, hasil sesuai. Program berjalan dengan sedikit kesalahan. Program tidak berjalan sempurna, banyak error. Program tidak dibuat atau tidak berjalan sama sekali.
Sikap Sangat aktif, bekerjasama, kreatif. Aktif dan bekerjasama dengan baik. Kurang aktif dan kadang tidak bekerjasama. Tidak aktif dan tidak bekerjasama.
